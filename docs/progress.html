UE5 Auth HTTP Client — 진행사항

UE5 클라이언트에서 FastAPI 인증 서버(http://localhost:8000)와 HTTP·JSON 통신을 구현하는 작업. 회원가입 · 로그인 · 내 정보 · 회원탈퇴 · 헬스체크.

● 현재 단계: C++ 소스 구현 완료 (Task 1~6) → 사용자 환경 빌드·검증 대기
UE 5.7 Blueprint-only → C++ 모듈 추가 HttpModule JWT Bearer

01확정된 결정

구현 방식
C++ 게임 모듈 신설 후 UE HttpModule 사용 (BP 기본 HTTP 노드 없음)
범위
5개 엔드포인트 전부 — signup / login / GET me / DELETE me / health
사용 방식
C++ 전용, BP 노출 안 함. 콘솔 명령 + 로그로 테스트
Base URL
기본값 http://localhost:8000 (변경 가능)

02대상 서버 엔드포인트

메서드경로인증성공
POST/auth/signup—201 / 409
POST/auth/login—200 → token
GET/users/meBearer200 / 401
DELETE/users/meBearer204 / 401
GET/health—200
